<하이퍼텍스트의 개념과 특징>

이란 웹 문서 양식을 지원하는 언어로 의미 그대로 문서의 특정 부분에 마크를 부여하면 컴파일러가 마크를 해석해 마크에서 지정하는 주소(URL)를 자동으로 연결하는 기능을 가지고 있 다. 특정 문서에서 바로 다른 문서로 옮겨가는 이 기능을 하이퍼링크 기능이라고 한다.
HTML로 작성되어 이런 하이퍼링크로 문서와 문서가 연결된 텍스트를 하이퍼텍스트 라고 한다.
특징으로는 비선형성 : 선형성을 근간으로 하는 전통적인 텍스트와 구별시켜 주는 대표적인 특성 둘째, 하이퍼텍스트는 역동적이다.
기존의 전통적인 텍스트와 달리 하이퍼텍스트는 텍스트, 이미 지 ,그래픽, 도표, 공식, 사운드 등 객체라 불리는 다양한 구성 요소들이 다양한 방식으로 결합되면 서 스크린 상에 다양한 모습을 보여 주게 된다. 셋째, 하이퍼텍스트의 상호작용성은 저자 또는 설계자와 독자 또는 사용자 사이에 상호작용이 일 어난다는 것이다. 또한 하이퍼 텍스트는 멀티미디어이다. 따라서 확장성을 갖는다고 할 수 있다.

<웹의 개념과 특징>

World wide web의 줄임말로 온라인상에서 다양한 정보(웹 페이지)가 서로 연결되어 있는 것을 웹 이라 합니다.
웹 페이지는 글, 그림, 동영상 등 수많은 정보를 담고 있고 링크로 각각의 웹페이지 들이 연결되어 있습니다.
웹 즉 인터넷 상에서 정보가 서로 얽혀있는 무형의 정보 네트워크 웹은 인터넷으로 할 수 있는 여러 서비스 중 하나입니다.
특징으로는 TV나 라디오 처럼 정보와 서비 스를 제공하기만 함 웹사이트에 자신이 올린 데이터를 움직이거나 활용 할 수 없음.

<웹 2.0의 개념과 특징>

웹 2.0이라는 개념이 등장하기 전까지 차세대 엡(NGWeb = Next Generation Web)을 뜻하는 말로 는 시멘틱웹, (인공지능을 통한 자동화가 강화된 웹).을 사용했다. Web 2.0은 닷컴 붕괴 이후의 살아남은 업체들을 분류하고, 새로 등장하고 성장한 업체들의 공통 들을 총칭하는 용어로 차세대 인터넷을 대표하는 용어로 새로운 비즈니스 동향으로 새로운 웹의 트렌드를 지칭하는 용어 데이터의 소유자나 독점자 없이 누구나 손쉽게 데이터를 생산하고 인터 넷에서 공유할 수 있도록 한 사용자 참여 중심의 인터넷 환경입니다. 인터넷 상에서 정보를 모아 보여주기만 하는 웹 1.0 과 달리 웹 2.0 은 제공된 데이터를 활용하여 사용자가 정보 생성에 참 여하고 가치를 부여함으로써 누구나 다양한 신규서비스를 생산합니다. 특징으로는 1. 플랫폼으로서 웹 (플랫폼 위에 있는 서비스는 자신이 원하는 대로 할 수 있다.)
